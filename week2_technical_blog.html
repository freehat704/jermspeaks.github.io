Retire floats. Here are Inline-Blocks and Blocks!

Date: May 3rd, 2014

For a series of elements, particularly div and lists, you can have them display inline, inline-block, or block. This is done in CSS and uses the display property. Let’s go into detail about each of these.

Inline

Elements can sit to the left or right of other elements that are the children of the parents of the original element. In a nutshell, if you have an image and you need it to be inline with the text rather than have it is its own block and take up a lot of white space, the inline property will help you out.

However, there are a few things it can’t do. It can’t have a width and height set and it will not respect top and bottom margins and padding.
